# llm-unit-test

Installation Instructions:
* Create Python environment.  Assuming 3.11 and Anaconda:
 - conda create -n llm-unit-test
 - conda activate llm-unit-test
 - conda install python=3.11
* Install dependencies:
 - make install
* Create or edit spreadsheets in tests/spreadsheets
* Run test suite:
 - make test
